# CashNova Expense Tracker

CashNova is a simple and user-friendly web-based expense tracker that helps users record and manage their daily income and expenses.

## Features
- Add income and expenses
- View total income and total expenses
- Track daily spending
- Clean and simple user interface
- Beginner-friendly project

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Purpose
This project was built to practice frontend web development skills and understand how JavaScript can be used to handle data and user interactions.

## How to Use
1. Open the project in your browser.
2. Enter income or expense details.
3. View the updated expense summary.

## Future Improvements
- Add local storage support
- Add charts for expense visualization
- Improve UI and responsiveness
